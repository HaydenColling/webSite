Setting Up Website

This website that you are currently on is hosted in Azure. I will show you how I set it up and created a custom domain. Firstly navigate to the Azure Portal. Continue to the Static Web App section. For my purpose of this website, a static website is all I need. You will need to create a Static Web App. I attached the website to a created Subscription and Resource Group that I have designated for this website. For my Deployment, I chose to link the web app to my GitHub account.



As easy as that you have set up a website. I wanted to create a custom domain and not have a funky link they give out.



I went to Namecheap and bought this domain that you are currently on haydencyber.info. Now that I have the domain, I had to link the web app to my domain. Navigating to the Custom domains tab. Click Add, and generate a TXT Record. Once generated copy the Value that is given and add the value to your custom domain.



Once Azure verifies that you own the domain, you can begin to create CNAME records. Once again create another record on Azure, but this time a CNAME with the desired host. On your domain provider add another record with the same host and link the value to the desired URL.
