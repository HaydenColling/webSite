Automated Nmap Scan

The idea of this project was to get some exposure to how to use and set up SIEMs. I decided to set up a free version of Splunk in my home lab. First, I created two Virtual Machines on my Proxmox server. Both virtual machines will be using Ubuntu. On my first server, I downloaded the enterprise version of Splunk. To monitor data from another machine, I needed to set up a forwarder. Now switching to the other virtual machine, I was able to download the Splunk Universal Forwarder. I needed to point the Forwarder to the Splunk server.

In my case, I wanted to monitor the Auth.log file. So, I added a monitor on the specific file path.

Now we can see Splunk is monitoring all actions in Auth.log.
