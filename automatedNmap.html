Automated Nmap Scan

For this project, I wanted to make a script that would constantly run a Nmap scan on all the devices on my network. Firstly, I imported the Nmap library. We ask the user for four different scan options. Scan once, scan and email results, continuous scan, and finally a deep scan.


We then use a scanner to scan the network. For my purpose, I have a subnet network range of /24 so in my scanner, I input 192.168.1.0/24. This lets the scanner know what network to scan. Then you give the arguments of what type of scan this is. In my case it is a simple ping sweep, therefore the arguments would be "-sn". If the user chooses to email the results, we use the smtplib library. In my case, I used gmails API key in order to send the results.


For the deep scan, the user is able to input a single IP address or multiple. We use the scanner instead of the simple "-sn" ping sweep argument. It will use the "-A" argument to see any information it can acquire. All the information is stored in a dictionary. You can find the documentation here Link . For my purpose, I output the hostname, OS, OS accuracy, OS family, type, and open ports.
